# Requerimientos

Para poder ejecutar este proyecto es necesario disponer de:
  * Node.js
  * Python 3 y tener bs4, requests y pymongo instaladas.
  * Docker (Linux) o Docker Desktop (Windows/MAC).
  * Metamask plugin u otra funcionalidad de wallet.
